### 安装pip3和pip

python2.7和python3.6都已经安装好了，但是没有pip，执行下面指令

```
sudo apt install python3-pip  //安装pip3
sudo apt install python-pip   //安装pip（对应Python2）
```

你也可以不装pip，但有的地方还是有python2.7的需求
